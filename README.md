# Website Peraturan

Prototipe Astro untuk website full-text peraturan energi Indonesia.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Prototype Scope

- Konten website awal: `UU Nomor 30 Tahun 2009 tentang Ketenagalistrikan`.
- Uji reverse prompt PDF: `UU Nomor 20 Tahun 2014 tentang Standardisasi dan Penilaian Kesesuaian`.
- Data canonical berada di `src/data`.
